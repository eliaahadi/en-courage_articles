# en-courage_articles
Full Stack CRUD application built with JavaScript/React/Redux/PSQL/NodeJS/Golang.

# Setup
- To use the project, run these commands. 
- Setup
```
$ npm install
```

- Seed articles data
```
$ npm run-script web
```

- Front end 
```
$ npm run-script web
```

- Back end 
```
$ npm run-script serve
```
